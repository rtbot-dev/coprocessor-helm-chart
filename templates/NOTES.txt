Thank you for installing the coprocessor chart.

Release name: {{ .Release.Name }}
Chart version: {{ .Chart.Version }}

This release renders the coprocessor runtime with:
- a StatefulSet containing rtbot-redis and the ingress/egress runtime
- a SQL bootstrap Job that runs after install/upgrade by default
- a demo bootstrap Job and demo publisher Deployment when `demo.enabled=true`
- ConfigMaps for SQL and Connect configuration
- a Service exposing the ingress endpoint
- optional egress wiring back to ThingsBoard when enabled

Quick checks:
- Review the rendered ingress Service and Connect configuration.
- If ThingsBoard already runs in this namespace as `thingsboard`, the default URL should work without changes.
- Otherwise, override `thingsboard.baseUrl` for cross-namespace or external deployments.
- Confirm SQL files, image tags, persistence size, and resource requests match your deployment.
- If using existing secrets, verify `thingsboard.existingSecret` or `*.extraEnvFrom` references resolve in the target namespace.
